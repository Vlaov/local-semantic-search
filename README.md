# Система семантического поиска по локальному хранилищу медиафайлов

## О проекте

Проект посвящён разработке локальной системы семантического поиска по медиафайлам.  
Система должна помогать пользователю находить изображения, видео и аудиофайлы не только по названию, но и по смыслу текстового запроса.

Например, пользователь может ввести:

- `фото с морем`
- `видео с концерта`
- `лекция по нейросетям`

и система попытается найти подходящие файлы, даже если в их названии нет этих слов.

---

## Проблема

Обычный поиск по локальному хранилищу в основном работает по имени файла.  
Это неудобно, потому что:

- многие медиафайлы имеют стандартные названия, например `IMG_1234.jpg`
- пользователь может помнить содержание файла, но не помнить его название
- поиск нужного файла в большом архиве занимает много времени

---

## Цель проекта

Разработать прототип локальной системы, которая сможет:

- индексировать медиафайлы в выбранной папке
- анализировать содержимое файлов
- выполнять поиск по текстовому запросу
- выводить пользователю наиболее подходящие результаты

---

## Основные функции

### MVP

- сканирование выбранной папки
- индексация изображений
- поиск по текстовому запросу
- вывод списка найденных файлов
- отображение пути к файлу
- сортировка результатов по релевантности

### Планируемые улучшения

- поддержка видео и аудио
- распознавание речи
- фильтрация по типу файла
- предпросмотр изображений
- графический интерфейс
- автоматическое обновление индекса

## UX/UI

Интерфейс проекта должен быть простым и понятным.  
Пользователь взаимодействует с системой через набор основных экранов.

### Основной сценарий работы

1. Пользователь выбирает папку с медиафайлами
2. Система сканирует файлы и создаёт индекс
3. Пользователь вводит текстовый запрос
4. Система показывает наиболее подходящие результаты
5. Пользователь открывает нужный файл

### Основные элементы интерфейса

- кнопка выбора папки
- кнопка запуска индексации
- поле ввода поискового запроса
- кнопка поиска
- список найденных файлов
- отображение пути к файлу
- фильтр по типу файла

### Экраны интерфейса

#### Экран выбора папки

Пользователь выбирает локальную папку, в которой будет выполняться поиск.

#### Экран индексации

Система показывает процесс обработки файлов и статус создания индекса.

#### Экран поиска

Пользователь вводит запрос, например:

`фото с морем`

#### Экран результатов

Система выводит список найденных файлов, где для каждого результата отображаются:

- имя файла
- тип файла
- путь к файлу
- степень соответствия запросу

### Прототипы
